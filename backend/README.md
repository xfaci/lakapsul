# La Kapsul Backend (Fastify + TypeScript)

## Scripts
- `npm run dev` – démarrage en watch avec tsx
- `npm run build` – compilation TypeScript
- `npm run start` – exécute le build compilé

## Structure
Voir `src/` pour la séparation config/controllers/routes/services/middlewares/sockets/utils`.

## Env requis
Voir `src/config/env.ts` pour la liste complète (JWT, Airtable, Stripe, OpenAI, n8n, etc.).
